# **App Name**: RupeeLedger

## Core Features:

- Account Management: Create, view, and manage multiple accounts such as Cash, Bank, or Savings, along with their initial balances. All values will be handled in Indian Rupees (INR).
- Credit/Debit Entry: Easily record credit (incoming) and debit (outgoing) transactions with details like amount, date, description, and chosen account.
- Transaction Ledger View: Access a chronological, filterable list of all transactions for a selected account, displaying credit, debit, and a running balance in INR.
- Balance Overview Dashboard: View real-time current balances for all individual accounts and a consolidated total balance in INR.
- Voucher Printing: Generate and print professional transaction vouchers for any recorded credit or debit entry.
- AI Narration Suggester: An AI tool that suggests concise and relevant narrations or particulars for new credit/debit entries based on transaction type, amount, or chosen account.

## Style Guidelines:

- Primary color: A deep, professional navy blue (#13315E) symbolizing trustworthiness and stability.
- Background color: A subtle, clean light blue-gray (#F2F5F7) ensuring excellent readability for numerical data and text.
- Accent color: A vibrant, clear cyan (#3DD6E8) used for calls to action and highlighting important information.
- Headline and body font: 'Inter', a modern sans-serif for its clarity, objectivity, and excellent readability across all financial data.
- Utilize clean, modern, line-based icons relevant to financial transactions, ledger management, and printing functions.
- Employ a structured, grid-based layout for transaction lists and dashboards, prioritizing clarity and ease of navigation through financial data.
- Subtle and functional micro-animations for feedback on user actions, like successful transaction recording or voucher generation, to enhance user experience without distraction.
